폰트 파일은 저작권 및 배포 관리를 위해 프로젝트 ZIP에 포함하지 않습니다.

아래 두 파일을 이 폴더에 직접 복사하세요.
- NanumSquareR.ttf
- NanumSquareB.ttf

최종 구조
assets/fonts/
├─ font_config.json
├─ NanumSquareR.ttf
└─ NanumSquareB.ttf

적용 방식
- 웹 미리보기: 위 TTF 파일을 읽어 나눔스퀘어를 적용합니다.
- PPT: Windows에 설치된 '나눔스퀘어' 글꼴명을 적용합니다.
- 파일이 없거나 글꼴이 설치되지 않은 PC에서는 대체 글꼴로 표시될 수 있습니다.

다른 글꼴을 쓸 때는 font_config.json의 세 값을 수정하세요.
