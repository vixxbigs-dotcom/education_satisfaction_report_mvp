# CHANGELOG v1.3

## UI 개선
- 흰 배경 콘텐츠 장표 스타일을 기준으로 교육 개요, 현장 사진, 통계 장표 UI를 조정했습니다.
- 주요 표, 문항 제목, 그래프 라벨, 주관식 응답 글자 크기를 전반적으로 키웠습니다.
- 교육 개요 표의 좌측 항목 열을 주황색 배경/흰색 글씨로 변경했습니다.
- 사진이 없는 현장 사진 장표는 회색 플레이스홀더 없이 빈 흰 배경으로 표시합니다.

## 객관식 그래프 개선
- 응답 수가 0인 항목은 0% 너비의 작은 네모가 표시되지 않도록 변경했습니다.
- 응답이 있는 항목만 컬러 막대를 표시합니다.

## 주관식 미리보기 동기화 개선
- 주관식 입력부에서 수정한 응답 내용과 미리보기/PPT에 출력되는 응답 내용이 달라지는 문제를 수정했습니다.
- 주관식 응답 수정 시 슬라이드 분할 계획이 즉시 갱신되도록 동기화했습니다.

## 장표 구성
- 설문 구성 장표는 글자 크기 확대에 맞춰 한 장당 최대 8문항 기준으로 분할합니다.
