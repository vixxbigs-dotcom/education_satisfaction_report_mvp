# v1.4 변경 사항

## 반영 내용

1. 표지 과정명에서 `[ ]` 기호를 제거했습니다.
2. 입력 설정 패널은 현재 장표에 실제로 반영되는 필드만 표시합니다.
   - 표지: 회사 이름, 교육 과정명, 교육 일정
   - 만족도 요약: 총 수강인원, 응답자 수
   - 목차·구분장·감사 장표: 입력 패널 숨김, 미리보기 전체 폭 사용
3. 객관식 문항 제목을 길이에 따라 자동 축소하고 차트 시작 위치를 조정하여 텍스트 겹침을 줄였습니다.
4. 응답 분포가 0인 항목은 HTML 미리보기와 PPT에서 색상 막대 도형을 생성하지 않습니다.
5. 흰 배경 장표 우측 하단 로고를 사용자가 제공한 멀티캠퍼스 공식 로고로 교체했습니다.
6. 표, 응답 라벨, 주관식 본문 등 기본 글자 크기를 전반적으로 확대했습니다.

## 기존 프로젝트에서 덮어쓸 파일

```text
app.py
src/preview_renderer.py
src/ppt_renderer.py
assets/multicampus_logo.png
README.md
docs/CHANGELOG_V1_4.md
```

덮어쓴 뒤 실행 중인 Streamlit을 `Ctrl + C`로 종료하고 다시 실행합니다.

```bash
python -m streamlit run app.py
```
