# V2 Beta 변경 사항

## 데이터 분석

- 1~5점, 1~7점, 0~10점 척도 자동 인식
- 단일선택과 복수선택 문항 모델 추가
- 구글폼 응답 시트 자동 분석
- 주관식 문장에서 쉼표가 사용된 경우 복수선택으로 잘못 분류하지 않도록 반복성과 고유값 비율을 함께 검증
- 실명 강사만 추출하고 `(강사)`, `퍼실리테이터(강사)` 같은 일반 표기는 이름으로 저장하지 않음
- 강사명에서 `교수`, `강사`, `코치`, `대표` 등 직함을 제거해 내부적으로 동일한 이름으로 관리

## 장표 구조

- 서로 다른 척도는 만족도 요약 장표를 분리해 각 척도 최대값에 맞는 축 사용
- 공통문항과 강사문항 분리
- 강사 2명 이상일 때 비교 표 자동 생성
- 강사별 상세 문항을 강사 단위로 묶어 생성
- 0~10점 문항은 세로 응답 분포 차트로 표시
- 단일선택은 편집 가능한 PowerPoint 원형차트 사용
- 복수선택은 가로막대와 응답자 대비 선택률 표시
- 종합 시사점 자동 초안 및 편집 기능 추가

## 유지 사항

- 추천 의향 문항은 평균과 0~10점 분포만 표시
- 추천자·중립자·비추천자·최종 추천지수는 생성하지 않음
- 기존 오렌지 디자인과 흰 배경 장표의 멀티캠퍼스 로고 유지
- 기존 5점 척도 엑셀과 전치형 엑셀 호환 유지
