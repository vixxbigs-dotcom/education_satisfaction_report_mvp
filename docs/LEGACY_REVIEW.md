# 기존 프로그램에서 참고한 기능

새 MVP는 기존 프로그램을 그대로 확장하지 않고 구조를 분리해 다시 구성했습니다.

## 재사용한 설계 개념

- 문자형 리커트 척도 자동 인식
- 주관식 ‘없음’류 답변 필터링
- `python-pptx` 기반 편집 가능한 PPT 생성
- 사진 6장 단위 자동 배치
- Streamlit 웹 배포 방식

## 새로 설계한 부분

- 전치형 엑셀과 일반형 엑셀을 모두 지원하는 구조 판별
- 여러 시트 중 설문 응답 시트 자동 선택
- 엑셀 업로드만으로 기본 입력값 자동 채움
- 공통 `ReportData`를 HTML과 PPT가 함께 사용
- 문항 수와 주관식 분량에 따른 동적 장표 생성
- 장표별 좌측 편집·우측 실시간 미리보기

## 제외한 기존 구조

- 별도 `설정.xlsx`를 필수로 요구하는 방식
- 강사마다 설문 파일을 따로 올리는 방식
- Windows PowerPoint COM에 의존하는 미리보기
- Tkinter GUI와 Streamlit UI가 중복된 구조
