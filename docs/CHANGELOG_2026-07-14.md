# 2026-07-14 수정 내역

## 1. 설문 문항 번호 중복 제거

원본 문항의 앞에 포함된 `1.`, `2.`, `1)`, `2)` 등의 번호를 제거합니다. 결과보고서에서는 설문 순서에 따라 `1.`, `2.`, `3.` 형태의 연속 번호를 새로 부여합니다.

예시:

- 수정 전: `2. 1. [AI 시대 리더십 특강] 과정에 만족하십니까?`
- 수정 후: `2. [AI 시대 리더십 특강] 과정에 만족하십니까?`

## 2. 주관식 결과 목록 변경

- 수정 전: 응답별 구분선
- 수정 후: `- 응답 내용` 형태의 불릿 목록

HTML 미리보기와 PPTX 모두 동일하게 반영됩니다.

## 3. 입력값 자동 반영

`src/live_input.py`와 `src/live_input_frontend/`에 로컬 Streamlit 컴포넌트를 추가했습니다. 텍스트를 입력하고 잠시 멈추면 별도 `Ctrl + Enter` 없이 미리보기가 갱신됩니다.

- 기본 지연 시간: 220ms
- 주관식 다중 입력: 280ms
- 외부 CDN 및 별도 로그인 불필요

## 4. 현재 슬라이드 유지

페이지 번호가 아니라 슬라이드 고유 ID를 상태값으로 보관합니다. 입력값 변경으로 Streamlit이 다시 실행되어도 현재 슬라이드에 머뭅니다. 주관식 답변 수 변경으로 동적 페이지가 사라지는 경우에는 동일 문항의 가장 가까운 페이지를 유지합니다.
