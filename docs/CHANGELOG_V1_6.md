# v1.6 Beta 변경사항

- 웹 상단 서비스명을 `교육만족도 결과보고서 생성기 (Beta)`로 변경했습니다.
- 긴 서비스명이 잘리지 않도록 제목 글꼴 크기, 줄 높이, 줄바꿈 규칙을 보정했습니다.
- 편집할 필요가 없는 구분장·목차·감사 페이지의 미리보기 폭과 높이를 소폭 축소했습니다.
- 편집 페이지의 입력부와 출력 미리보기 사이에 드래그 가능한 세로 분할선을 추가했습니다.
  - 입력부 최소 폭: 27%
  - 입력부 최대 폭: 68%
  - 브라우저 세션 동안 마지막 너비를 기억합니다.
- `PPT 생성 / 최신 내용 반영` 기본 버튼을 멀티캠퍼스 주황색 `#ED702C`로 변경했습니다.
- Python 패키지 의존성 변경은 없습니다.
