# v1.2 디자인 반영 내역

## 멀티캠퍼스 결과보고서 UI 적용

- 표지, 목차, 섹션 구분장, 마지막 장표에 제공된 회색/흰색 분할 배경을 적용했습니다.
- 과정명, 섹션명, 감사 문구는 배경 이미지 위에 편집 가능한 텍스트로 생성됩니다.
- 일반 장표의 제목, 구분선, 우측 섹션 태그를 주황색 결과보고서 스타일로 통일했습니다.
- 우측 하단 멀티캠퍼스 로고를 반영했습니다.

## 만족도 그래프 개선

- 만족도 요약을 0~6점 축의 세로 막대그래프 형태로 변경했습니다.
- 한 장에 최대 7개 문항을 표시하며 초과 문항은 다음 요약 장표로 자동 분할됩니다.
- 문항별 결과는 응답자 수를 기준으로 한 가로 막대그래프로 표시합니다.
- 응답 수와 축 눈금, 문항 라벨을 모두 편집 가능한 PPT 도형과 텍스트로 생성합니다.

## 주관식 결과 개선

- 응답을 구분선 없이 `- 응답 내용` 형식으로 배치합니다.
- 답변이 많거나 길면 장표가 자동 분할되며, 글자 수에 따라 글자 크기를 조정합니다.

## 미리보기와 PPT 페이지 일치

- 설문 구성과 만족도 요약의 동적 분할 장표를 `slide_plan.py`에서 한 번만 계획합니다.
- 웹에서 이동 가능한 모든 장표와 실제 PPT에 생성되는 장표의 순서를 일치시켰습니다.
