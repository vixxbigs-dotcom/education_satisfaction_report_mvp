# V2.1 만족도 요약 개선

## 변경 사항

- 만족도 요약 장표 제목에서 `1~5점 척도`, `0~10점 척도` 문구를 제거했습니다.
- 서로 다른 척도는 축 정확성을 위해 내부적으로 계속 분리하되, 화면과 PPT에는 척도명을 노출하지 않습니다.
- 0~10점 및 1~10점처럼 점수 구간이 넓은 문항은 만족도 요약에서 평균 막대 대신 점수별 응답 분포를 표시합니다.
- 각 점수별 응답 인원과 비율을 함께 표시합니다.
- 전체 응답자 수와 해당 문항의 유효 응답자 수가 다를 경우 둘 다 표시합니다.
- 웹 미리보기와 실제 PPT에 동일한 장표 구성을 적용했습니다.
